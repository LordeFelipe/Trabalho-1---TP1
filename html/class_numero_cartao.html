Descrição detalhada

Classe do tipo NumeroCartao.

Classe que armazena o número do cartão de crédito do usuário e contém os métodos de manipulção desse valor. O Cartao tem o formato de uma string com 16 digitos, contendo apenas números, e que respeita o algoritmo de Luhn.
