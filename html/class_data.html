Descrição detalhada

Classe do tipo Data.

Classe que armazena uma data e contém os métodos de manipulação desse valor. A data tem o formato de uma string de tamanho 11 no formato dia/mes/ano onde o dia tem 2 caracteres, o mês 3 e o ano 4.
