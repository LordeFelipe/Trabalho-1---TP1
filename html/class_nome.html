Classe do tipo Nome.

Classe que armazena uma string com o nome do usuário e contém os métodos de manipulção desse valor. O nome tem o formato de uma string com no máximo 15 digitos, contendo ao menos uma letra, sem espaços seguidos e com todos os pontos sucedendo letras.
