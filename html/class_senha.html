Descrição detalhada

Classe do tipo Senha.

Classe que armazena uma string com a senha do usuário e contém os métodos de manipulção desse valor. A senha tem o formato de uma string com 8 digitos, contendo ao menos uma letra maiuscula, minuscula, um número e um digito(! # $ % &) e sem caracteres repetidos.
